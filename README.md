# 雀成績 — 雀荘実績記録アプリ

雀荘での実績（半荘ごとの精算pt・着順・チップ・時給）を記録する、プレイヤー視点の成績管理アプリ。

- **スタック**: バニラ JavaScript（ビルド不要の静的3ファイル）
- **データ**: ブラウザ内 `localStorage`（将来的に Supabase 同期を予定）
- **対象**: スマホでの利用を想定（将来 Capacitor でアプリ化）

## 機能

- **店設定**: 店ごとにルール（配給原点・返し点・レート・ウマ・オカ・チップ単価・ゲーム代・トップ賞）を登録
- **来店記録**: 半荘中にチップの増減を +/- ボタンで記録 → 半荘終了時に着順と持ち点を入力 → pt変動を自動計算しデポジット（累計収支）に加算
- **時給・時間表示**: 経過時間と pt 時給を表示
- **半荘の編集・削除**: 記録済みの半荘をタップして修正・取り消し
- **成績**: 通算収支・平均着順・トップ率・連対率・ラス率・店別収支

## pt 計算式

```
半荘pt = (素点 + オカ + ウマ) × レート + チップ×単価 − トップ賞(1位のみ) − ゲーム代
  素点 = (持ち点 − 返し点) / 1000
  オカ = (返し点 − 配給原点) × 4 / 1000   （1位のみ・自動加算ON時）
  トップ賞 = 1位が支払う（減点）
```

## ローカルで動かす

```bash
python3 -m http.server 4137
# http://localhost:4137 を開く
```
